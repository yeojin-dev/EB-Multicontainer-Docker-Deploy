# EB-Multicontainer-Docker-Deploy
AWS Elastic Beanstalk Multicontainer Docker 배포 예제

## Requirements

#### Product

- Python (3.7)
- pipenv
- Django (2.x)

#### Dev

- awsebcli

## Running

#### Local

```shell
# Move project's directory
docker-compose up -d
```

## Deploying

```shell
# Move project's directory
eb deploy
```
